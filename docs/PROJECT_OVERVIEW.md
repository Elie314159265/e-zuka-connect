# E-Zuka Connect プロジェクト概要

## プロジェクトビジョン

E-Zuka Connectは、福岡県飯塚市の商店街が直面する来訪者減少や売上低迷といった課題を解決するため、「データ駆動型イベント・店舗支援プラットフォーム」です。

### コンセプト

地域の個店と生活者をデータで繋ぎ、効果的な販売促進活動とイベント開催を支援することで、商店街全体の活性化に貢献します。

### 解決する課題

- 商店街の来訪者減少
- 各店舗の売上低迷
- イベント効果の測定困難
- プロモーションプラットフォーム不足
- その日の環境に適した商品のプロモーション

## アプリケーション概要

### 店舗側の機能

- セール情報やイベントの簡単登録
- データに基づく顧客分析
- 売上・購買動向の可視化
- AI経営アドバイス

### 利用者側の機能

- クーポン取得
- レシート登録によるポイント獲得
- 地域貢献度の可視化
- イベント情報の閲覧

### データ分析機能

定期的に市や気象庁のオープンデータを取得し、以下を実現：

- 人の行動や購買動向の分析
- アプリの利用状況とオープンデータの掛け合わせ
- 「雨の日は主婦層がこの商品を求めている」といったインサイトの抽出
- ダッシュボード化して店舗側に提供

## プロジェクトの世界観

「データで繋がり、ともに成長する新しい商店街の形」を実現します。

- 多くの店舗がデータを収集・集約化
- 定期的なデータ分析によりその日の環境に最適な商品を提案
- 利用者はレシート送信で貢献ポイントを獲得
- 地域全体の貢献ポイント達成で記念セールやクーポン発行
- 地域全体で目標を達成する一体感の創出

## 主要機能

### 1. レシートOCR機能
- Google Document AIを使用した高精度なレシート読み取り
- スマホでレシートを撮影するだけでデータ化
- GCS (Google Cloud Storage)への安全な保存

### 2. ゲーミフィケーション
- ポイント・バッジ・レベルシステム
- 地域貢献度の可視化
- ユーザーエンゲージメント向上

### 3. データ分析
- 天候データと売上データの相関分析
- 顧客層分析（年齢・性別）
- 商品ランキング
- トレンド予測

### 4. AI経営アドバイス
- Google Gemini APIを活用
- データに基づく経営アドバイス提供
- 売上予測

### 5. プロモーション管理
- クーポン・イベント管理
- 店舗向け販売促進ツール

## 技術的特徴

このプロジェクトは以下の技術的特徴を持ちます：

- **高可用性**: Google Kubernetes Engine (GKE)を利用
- **スケーラビリティ**: イベント時の高アクセスに耐えるインフラ
- **マイクロサービス**: 独立したサービスで開発速度向上
- **自動化**: CronJobによる定期的なデータ取得・分析
- **セキュリティ**: HTTPS化、JWT認証、データ暗号化
